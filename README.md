## 소개

Workmap은 여러 작업을 오가는 사람을 위한 데스크톱 작업 런처입니다. 자주 쓰는 폴더·파일·앱·링크·스크립트를 한 화면의 카드로 모아두고, 클릭 한 번으로 작업 환경에 바로 들어갑니다.

## 주요 기능

- **원클릭 실행** — 폴더·파일·앱·URL·스크립트를 카드로 등록해 바로 열기
- **시각적 정리** — 진행도·체크리스트·상태·미니보드로 흩어진 작업을 한눈에
- **자유로운 커스터마이즈** — 보드 색·스티커·테마로 내 작업실처럼 꾸미기
- **로컬 우선** — 데이터를 서버에 올리지 않고 내 PC에 저장
- **다국어 UI** — 한국어·영어·일본어 등 지원

## 추천 대상

- 만화·영상·음악·게임 등 여러 프로젝트를 병행하는 창작자
- 폴더·링크·자료가 흩어져 매번 다시 찾는 사람
- 무거운 협업 도구 없이 내 작업 흐름만 가볍게 정리하고 싶은 사람
