## 무엇을 해주나요

- 여러 작업을 오갈 때 "지금 뭘 해야 하지?"에서 새는 시간을 줄여줍니다.
- 폴더·파일·앱·URL·스크립트를 카드로 만들어 **원클릭 실행**.
- 진행도·체크리스트·미니보드로 흩어진 작업을 한눈에 정리.
- 색·스티커·테마로 내 작업실처럼 꾸미고, 데이터는 서버가 아니라 **내 PC**에 둡니다.

## 누구에게 좋나요

- 만화·영상·음악·게임 등 여러 프로젝트를 병행하는 창작자
- 폴더·링크·자료가 흩어져 매번 다시 찾는 사람
- 무거운 협업 도구 없이 내 작업 흐름만 가볍게 정리하고 싶은 사람

## 다운로드

[**Releases**](../../releases/latest)에서 최신 설치파일을 받아 실행하세요. 설치 후 시작 메뉴에서 **Workmap**을 엽니다.

> 중요한 보드는 `JSON 내보내기`로 따로 백업해두는 것을 권장합니다.
